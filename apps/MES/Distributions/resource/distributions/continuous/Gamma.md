$\text{Gamma Distribution: } X \sim Gamma(\alpha, \lambda)$

* PDF and Support: $f(x) = \frac{1}{\Gamma(\alpha)} (\lambda x)^{\alpha} e^{-\lambda x} \frac{1}{x}, \ x \in (0, +\infty)$
* mean: $\frac{\alpha}{\lambda}$
* Variance: $\frac{\alpha}{\lambda^2}$
